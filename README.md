# Loja Online AGENS FITNESS

E-commerce completo para a academia AGENS FITNESS, desenvolvido em HTML, CSS e JavaScript puro.

## Funcionalidades

- ✅ Design responsivo e mobile-first
- ✅ Carrinho de compras com localStorage
- ✅ Filtros de produtos por categoria
- ✅ Controle de estoque em tempo real
- ✅ Integração com WhatsApp para pedidos
- ✅ Menu hamburguer para mobile
- ✅ Animações suaves e transições
- ✅ Notificações toast
- ✅ Identidade visual completa da AGENS FITNESS

## Estrutura do Projeto

```
loja-agens/
├── index.html          # Arquivo principal com todo o código
└── LICENSE             # Licença do projeto
```

## Como Usar

1. Abra o arquivo `index.html` em qualquer navegador moderno
2. Navegue pelas categorias de produtos
3. Use os filtros para encontrar produtos específicos
4. Adicione produtos ao carrinho
5. Clique no ícone do carrinho para visualizar os itens
6. Finalize o pedido via WhatsApp

## Produtos Incluídos

- **Suplementos**: Creatina, Pré-treino, Whey Protein
- **Roupas**: Camisetas dry fit, Shorts, Tops
- **Acessórios**: Straps, Coqueteleira, Boné, Toalha, Squeeze

## Tecnologias Utilizadas

- HTML5 semântico
- CSS3 com variáveis CSS e Grid/Flexbox
- JavaScript ES6+ (vanilla)
- Google Fonts (Rajdhani e Exo 2)
- localStorage para persistência de dados

## Responsividade

- Desktop: Layout completo com grid de produtos
- Tablet: Ajustes para telas médias
- Mobile: Menu hamburguer e layout single-column
